# 🐂 BULL SUPPLEMENTS
### Dark E-commerce Landing Page

**Live Demo:** https://terry-stang92.github.io/bull-suplements/

---

## Overview
Production-ready dark-themed e-commerce landing page built for a supplements brand. Designed to be adapted for any fitness, sports, or supplement business. Clean code, modern design, fully responsive.

---

## Features
- ✅ Announcement bar with auto-slider
- ✅ Sticky navbar — changes on scroll
- ✅ Dark / Light mode toggle with localStorage
- ✅ Hero section with CTA buttons
- ✅ Stats bar — 50K+ athletes, 4.9 rating
- ✅ Products grid with category filters
- ✅ Shop by Goal — 4 image cards with hover effects
- ✅ Why Choose Us — 3 feature cards
- ✅ Testimonials — 3 customer reviews
- ✅ Newsletter form with validation
- ✅ Footer with 4 columns + social links
- ✅ Payment method icons
- ✅ WhatsApp floating button
- ✅ Scroll animations — fade in on scroll
- ✅ Fully responsive — mobile first
